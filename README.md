# FrankRPG

_A small RPG project to deepen my understanding of Python programming._

## 📖 About

FrankRPG is a simple, text-based role-playing game (RPG) developed as a personal learning project. It is designed to help me practice Python fundamentals such as:

- Object-Oriented Programming (OOP)
- File handling
- Game loops and logic
- User input and interaction
- Modular code structure

This project is now public to share my learning journey and possibly help others who are also learning Python.

## 🚀 Features

- Character creation and stats
- Basic combat system
- Inventory management
- Simple quests or encounters

## 🛠️ Technologies Used

- Python 3.x
- Standard Python libraries (no external dependencies)
